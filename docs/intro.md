# Headless Shopify Using Next.js

Welcome to the **Headless Shopify** documentation!

Open-source Next.js Shopify storefront starter for building headless ecommerce experiences fast.

## Live Demo

Check out the live demo: [https://headless-shopify-site.vercel.app](https://headless-shopify-site.vercel.app/)

## What is Headless Shopify?

This is a modern, production-ready boilerplate for building headless Shopify storefronts using Next.js. It provides a complete foundation for creating custom ecommerce experiences with Shopify's powerful backend.

## Key Features

- **Next.js v16.1.6** with App Router
- **Shopify Storefront API** integration
- Complete pages: Home, PLP, PDP, Search, Signup, Login, Cart, Account
- Cart creation and line-item updates
- GraphQL queries and mutations via `.graphql` files
- **Better-Auth** authentication
- **Zod** validation for type-safe data handling

## Tech Stack

- **Next.js** v16
- **React** 19
- **TypeScript**
- **Tailwind CSS**
- **Shopify Storefront API**
- **better-auth**
