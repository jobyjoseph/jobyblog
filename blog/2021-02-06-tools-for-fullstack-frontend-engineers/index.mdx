# Tools for Fullstack Frontend Engineers

As a fullstack JavaScript developer, I come across different technologies and tools on a daily basis. This is a collection of such tools that might come handy for you as well.

- [Express](https://expressjs.com/) - A Nodejs framework to create websites and APIs easily.
- [ExpressJS Async Errors](https://www.npmjs.com/package/express-async-errors) - NPM package to enable Express framework to handle asychronous errors.
- [Express Validator](https://express-validator.github.io) - Validate or sanitize API requests received by an Express application.
- [Helm](https://helm.sh/) - Package manager to find, share and use software built for Kubernetes.
- [Istio](https://istio.io/) - Platform independent service mesh that comes handy in microservice development.
- [Jest](https://jestjs.io/) - JavaScript testing framework.
- [Mongoose](https://mongoosejs.com/) - NPM package to communicate with MongoDB.
- [Node.js](https://nodejs.org) - A software using which we can run JavaScript in a server.
- [Postman](https://www.postman.com/) - Software to test APIs.
- [Supertest](https://www.npmjs.com/package/supertest) - NPM package used for testing purpose to send fake requests to APIs.
- [TypeScript](https://www.typescriptlang.org/) - Helps to define data type of a variable while writing JavaScript.
